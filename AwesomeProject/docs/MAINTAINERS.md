# Maintainers

This document records lightweight maintenance expectations for the project.

## Release Checklist

- Confirm `npm run lint`, `npm run typecheck`, and `npm test` pass.
- Confirm the Android debug build passes.
- Update `CHANGELOG.md`.
- Tag the release from `main`.

## Triage Guidelines

- Label reproducible bugs clearly.
- Ask for environment details when build or device behavior is unclear.
- Prefer small pull requests that are easier to review and revert.
